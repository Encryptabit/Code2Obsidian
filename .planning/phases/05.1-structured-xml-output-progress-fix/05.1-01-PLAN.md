<task type="auto">
  <name>Task 2: XML parser, enricher integration, and emitter consumption</name>
  <files>
    Enrichment/LlmEnricher.cs
    Emission/ObsidianEmitter.cs
  </files>
  <action>
    1. In Enrichment/LlmEnricher.cs, add an internal static method:
       ```csharp
       internal static EnrichmentResponse? ParseXmlResponse(string raw)
       ```
       Implementation:
       a. Return null if input is null/whitespace
       b. Try XDocument.Parse($"<root>{raw}</root>") to extract <summary>, <purpose>, <tags> elements
       c. On XmlException: fallback to regex extraction with RegexOptions.Singleline:
          - <summary>(.*?)</summary>
          - <purpose>(.*?)</purpose>
          - <tags>(.*?)</tags>
       d. Apply WebUtility.HtmlDecode to extracted values
       e. Parse tags by splitting on comma, then trim each
       f. Double fallback: if neither XDocument nor regex finds content, wrap raw text as
          new EnrichmentResponse(raw, raw, Array.Empty<string>())

    2. In Enrichment/LlmEnricher.cs ProcessMethodAsync and ProcessTypeAsync:
       a. After var summary = response.Text, call ParseXmlResponse(summary) to get EnrichmentResponse
       b. Pass EnrichmentResponse to SummaryCache.Put(entityId, hash, enrichmentResponse, modelId)
       c. Store EnrichmentResponse in the EnrichedResult dictionary (not raw string)
       d. If ParseXmlResponse returns null, increment the failed counter and skip storage

    3. In Emission/ObsidianEmitter.cs, change three render methods' string? summary parameter
       to EnrichmentResponse? enrichment:
       - RenderMethodNote
       - RenderClassNote
       - RenderInterfaceNote

       In each method:
       a. ## Summary section: render enrichment.Purpose as a bold line, then enrichment.Summary as body
       b. Merge enrichment.Tags into YAML frontmatter tags: list, prefixed with "llm/"
       c. Tag sanitization (LLM-07): each tag is lowercased, whitespace trimmed,
          non-alphanumeric chars (except -) stripped, empty tags discarded
       d. EmitAsync TryGetValue calls stay the same (dict key is still string)
  </action>
  <verify>
    dotnet build 2>&1 | findstr /i "error"
    Expect: clean build, zero errors.
  </verify>
  <done>
    ParseXmlResponse handles well-formed XML, malformed XML (regex), garbage (double fallback), null (returns null).
    ProcessMethodAsync/ProcessTypeAsync store EnrichmentResponse objects (not strings).
    ObsidianEmitter renders purpose + summary in ## Summary section.
    LLM tags appear in frontmatter prefixed with llm/ and sanitized.
    Full build succeeds.
  </done>
