       - Format: "Namespace.ClassName"
       - Include a static factory method FromSymbol(INamedTypeSymbol) that generates the stable string

    4. Create Analysis/Models/MethodInfo.cs:
       - A sealed record (NOT a Roslyn symbol -- pure domain model)
       - Properties: MethodId Id, string Name, string ContainingTypeName, TypeId ContainingTypeId, string FilePath, string DisplaySignature, string? DocComment
       - No references to Microsoft.CodeAnalysis types anywhere in this file

    5. Create Analysis/Models/CallGraph.cs:
       - A sealed class containing:
         - Dictionary<MethodId, HashSet<MethodId>> CallsOut (caller -> callees)
         - Dictionary<MethodId, HashSet<MethodId>> CalledBy (callee -> callers) -- computed as reverse of CallsOut
       - Methods: void AddEdge(MethodId caller, MethodId callee), IReadOnlySet<MethodId> GetCallees(MethodId), IReadOnlySet<MethodId> GetCallers(MethodId)
       - Returns empty set (not null) when no edges exist for a given method

    6. Create Analysis/AnalysisResult.cs:
       - A sealed class (immutable after construction)
       - Properties: IReadOnlyDictionary<MethodId, MethodInfo> Methods, CallGraph CallGraph, int ProjectCount, int FileCount
       - Constructor takes all properties

    7. Create Analysis/AnalysisResultBuilder.cs:
       - A mutable builder that accumulates methods and call edges during analysis
       - Methods: void AddMethod(MethodInfo), void AddCallEdge(MethodId caller, MethodId callee), void IncrementProjectCount(), void IncrementFileCount()
       - Method Build() returns a sealed AnalysisResult

    8. Create Analysis/IAnalyzer.cs:

    9. Create Enrichment/IEnricher.cs:

    12. Create Emission/EmitResult.cs:
